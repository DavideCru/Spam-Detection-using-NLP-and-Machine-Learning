# Spam-Detection-using-NLP-and-Machine-Learning

Project Type: Machine Learning & Text Classification Project <br>
Tools Used: Python, pandas, scikit-learn, TF-IDF, Logistic Regression, Matplotlib <br><br>

This project develops a supervised machine learning model to classify SMS/email messages as Spam (0) or Ham (1) using Natural Language Processing techniques. <br>
The objective was to build an interpretable and statistically sound classifier capable of detecting spam while minimizing false positives. <br><br>

🔎 Methodology
• Data preprocessing and label encoding (Spam = 0, Ham = 1) <br>
• Exploratory analysis highlighting class imbalance <br>
• Text vectorization using TF-IDF <br>
• Classification using Logistic Regression <br>
• Model evaluation through accuracy and confusion matrix <br><br>

📊 Key Results
• 97% accuracy on the test set <br>
• Zero false positives (no legitimate messages incorrectly flagged as spam) <br>
• ~24% of spam messages misclassified as ham (room for recall improvement) <br>
• Clear lexical separation between spam and legitimate messages identified through coefficient analysis <br><br>

🧠 Conclusion
The model achieves strong performance while maintaining interpretability. <br>
Results highlight the importance of evaluating classification models beyond accuracy, especially in imbalanced datasets. <br>
This project demonstrates NLP feature engineering, supervised learning, model evaluation, and statistical reasoning in applied text classification. <br>
